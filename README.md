# Онлайн-курсы: Система управления курсами

## Описание
Консольная система на Java для онлайн-курсов. Сущности: Курс, Студент, Преподаватель.  
Операции: поиск студентов на курсе; фильтрация курсов по продолжительности.

## Установка
1. git clone <URL>.
2. javac *.java
3. java Main

## Структура
- Course.java
- Student.java
- Teacher.java
- DataManager.java
- Main.java
